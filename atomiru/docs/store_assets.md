# ストア用素材メモ

## 必要な素材

- アプリアイコン
- スクリーンショット
- サブタイトル
- 説明文
- キーワード
- プライバシーポリシー URL
- サポート URL
- リリースノート

## アイコン

- 1024x1024 の元画像を用意する
- 文字に依存しないシンプルなブランド表現を優先する
- アプリアイコンを更新した場合は `dart run flutter_launcher_icons` を再実行する

## スクリーンショット候補

- Inbox
- Share 保存画面
- Item 詳細
- Settings
- Paywall
- 履歴 / 一括操作
- Pro の通知設定

## ストア文言で必要なもの

- 英語 default listing のアプリ名: Catch Up Later
- 日本語 localized listing のアプリ名: あとみる
- サブタイトル案
- 1 行説明
- 長い説明文
- キーワード
- リリースノート
- Play Console の App access 用 Pro 確認手順

## URL

- Privacy Policy URL
- Terms URL
- Support URL
- Marketing URL（必要なら）

## 補足

- iOS と Android で必要なスクリーンショットサイズが異なる
- `atomiru_pro` の説明文もストア側で必要
- リリース記念価格は固定金額を書かず、必要なら「期間限定価格」と表現する
- 検索 UI は現リリースでは非表示のため、スクリーンショット素材には含めない
- 英語スクリーンショットでは `Catch Up Later` 表記に揃える
- 日本語スクリーンショットでは `あとみる` 表記に揃える
- 通知 / Share / 課金は実機確認後に素材を撮る方が安全
