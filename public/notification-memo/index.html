Concept

見返すためのメモから、目に入るメモへ。

やること、買うもの、あとで返す連絡。小さな用事ほど、
メモ帳の奥にしまうと忘れやすくなります。

通知にメモ。は、整理するためのノートではなく、
思い出すための置き場所です。
いつもの通知欄に残るから、特別に見返そうとしなくても気づけます。

WRITE IT. PLACE IT. NOTICE IT. WRITE IT. PLACE IT. NOTICE IT. WRITE IT. PLACE IT. NOTICE IT.

Flow

思いついたら、3つの流れで。

  1. 01

    書く

    タイトルだけでも、本文だけでも。
    忘れたくないことを短く残します。

  2. 02

    置く

    通知メモとして作成すると、いつもの通知領域にそのまま表示されます。

  3. 03

    気づく

    スマホを見たタイミングで自然に目に入り、終わったら消すだけです。

Scenes

日常の、少しだけ忘れたくないことに。

Morning

明日のごみ出し

朝に見たい用事を、夜のうちに通知へ置いておけます。

Move

折り返し連絡

移動中に思い出した用件を、落ち着くまで一時置きできます。

Shop

買い物の一言

シャンプー、電球、封筒。短い買い忘れ防止に向いています。

Before

外出前の持ちもの

鍵、書類、充電器など、家を出る前に見たいことを残せます。

Screens

入力、通知、一覧。
必要な画面だけ。

複雑に管理するより、すぐに書いて、通知で気づくことを優先した画面構成です。残したメモは一覧からも確認できます。

FAQ

使う前に気になること。

どんなメモに向いていますか？

買い物、折り返し連絡、持ち物、明日の用事など、短く残してあとで思い出したいメモに向いています。

長い文章の管理にも使えますか？

長文整理よりも、通知領域で自然に思い出すための一言メモに向いています。

残したメモはあとで確認できますか？

通知だけでなく、アプリ内の一覧画面から残したメモを見返せます。

安心して使うための情報はありますか？

フッターからプライバシーポリシーと利用規約を確認できます。
